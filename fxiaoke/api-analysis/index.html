所属字段说明
fieldInfoListobjectId每一个 objectsAndFields 下面的 fieldInfoList 的每个数组的 objectId 都是相同的，并且和 businessObjects 下面的 objectId相对应的，注意：上图是有客户，相关团队，人员三个分类，但是 businessObjects 只有两个分类
fieldInfoListfieldLocation猜测是字段的位置，可能是用来排序, 同一个 objectId 下面的相同 fieldType 的 fieldLocation 还是不相同的
fieldInfoListfieldType字段类型，目前看到的有： String , Number , Date , Boolean
fieldInfoListsubFieldTypeui 字段类型：当 ui 下面的 type 为 UI_MultiSelect 的时候，那么值就为：MultiSelectEnum。2. 当 ui 下面的 type 为 UI_Input 的时候，那么值就为：空字符串。3. 当 ui 下面的 type 为 UI_Input和 UI_Lookup（比UI_Input多这几个字段 "refObjName" "refKeyField" "refTargetField" "urlObj"）的时候，那么值就为：空字符串。4. 当 ui 下面的 type 为 UI_Selection 的时候，那么值就为：Circle，当是人员选择是这样的时候 。5. 当 ui 下面的 type 为 UI_Time 的时候，那么值就为：DateTime（fieldType 为 DateTime）或者空字符串（fieldType 为 Date）。6. 当 ui 下面的 type 为 UI_MultiSelect 并且 justLeafNodeSelect 为 1 的时候，那么值就为：LevelEnum ，意思就是做多级联动的时候使用这个。
